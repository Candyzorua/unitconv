# unitconv

First OCaml toy project that provides a CLI to convert units

## Learning topics:
- Functors
- First-class modules
- Generalized algebraic data types
- Command-line parsing with the `Command` library
- Error handling with `Or_error`
- Lazy computation with `lazy`
- Hashtables with `Hashtbl`
- `ppx_jane` annotations such as `@@deriving sexp`
